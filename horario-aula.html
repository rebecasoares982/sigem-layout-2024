Aluno: ALUNO RECUPERACAO SEMESTRAL


ENTENDA OS TIPOS DE TRANSFERÊNCIA:

TIPO 1 - TRANSFERÊNCIA DE ALUNO DENTRO DA PRÓPRIA ESCOLA: Este tipo de transferência irá alterar o status das disciplinas atuais para (7 - Transferido Turma) e o tipo de matrícula será alterada para matrícula aberta. Após isto a transferência se dará com a inclusão de matérias de outra turma na própria tela de matrícula para este aluno.

TIPO 2 - TRANSFERÊNCIA DE ALUNO PARA PRÓPRIA/OUTRA REDE MUNICIPAL: Este tipo de transferência irá remover o vínculo do aluno com a escola. Este aluno ficará disponível no menu "Acadêmico/Aluno/Alunos sem escola" para nova matrícula em qualquer outra escola. A matrícula anterior ainda permanecerá na escola antiga com status de transferido (6 - Transferido Escola) para que suas informações possam ser recuperadas sempre que solicitadas, a título de histórico escolar.
